# Analisis de Datos

## Metricas Clave:
- Distancia recorrida diaria por vehículo
- Número de vehículos con distancia recorrida mayor a 100 km
- Promedio de distancia recorrida por vehículo

## Insights:
- El script está generando un reporte diario de distancia recorrida para múltiples vehículos.
- El script está utilizando la API de GPS para obtener los datos de ubicación de los vehículos.
- El script está generando un reporte PDF diario con la información de distancia recorrida de cada vehículo.
- El script está enviando el reporte PDF diario a los responsables a través de correo electrónico.

## Visualizaciones Sugeridas:
- Gráfico de barras para mostrar la distancia recorrida diaria por vehículo.
- Gráfico de líneas para mostrar el promedio de distancia recorrida por vehículo a lo largo del tiempo.
- Tabla para mostrar la información de distancia recorrida de cada vehículo.
